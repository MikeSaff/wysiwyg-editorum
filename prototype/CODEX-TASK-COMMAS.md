## Проблема
Формула (2) неправильно делится на строки по запятым.

### Как сейчас (неправильно):
```
d/dt x(t) = f(x(t)) + D(x(t))w(t) + B(x(t))u(t),
x(t₀) = x₀,
y(t) = Cx(t),
u(t) ∈ U,
w(t) ∈ W,
t ∈ [t₀, tf]
```

### Как должно быть (из оригинала Word):
```
\begin{cases}
\frac{d}{dt} x(t) = f(x(t)) + D(x(t))w(t) + B(x(t))u(t), \\
x(t_0) = x_0, \\
y(t) = Cx(t),
\end{cases}
```
И отдельно: `u(t) \in U`, `w(t) \in W`, `t \in [t_0, t_f]`

## Правила для запятых в математике

1. Запятая **остаётся В КОНЦЕ строки** (не переносится на начало следующей)
2. НЕ разделять внутри аргументов функций: `f(x, y)` — запятая между x и y не является разделителем
3. Если формула — система уравнений в `\begin{cases}` — каждое уравнение отдельно, с запятой в конце
4. Если формула — одно уравнение с пояснениями — пояснения под формулой, не в формуле
