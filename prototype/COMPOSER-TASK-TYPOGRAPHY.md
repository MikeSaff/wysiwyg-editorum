# COMPOSER TASK: Автозамены по ГОСТ Р 7.0.110-2025

## ПРАВИЛО: после каждого изменения пиши в .context/progress.md

## Задача
Добавить ProseMirror input rules для автоматических типографических замен по ГОСТу.

## Что нужно сделать

### 1. Кавычки (§16.3)
При вводе `"` автоматически заменять на «ёлочки»:
- Открывающая `"` → `«`
- Закрывающая `"` → `»`
- Внутри уже открытых «» → „ и "

### 2. Тире (§16.1-16.2)
- `--` (два дефиса) → `–` (среднее тире, для диапазонов: 10–15)
- `---` (три дефиса) → `—` (длинное тире, для предложений)
- Дефис остаётся дефисом

### 3. Неразрывные пробелы (§9.4-9.5) — ОПЦИОНАЛЬНО
При вводе автоматически вставлять неразрывный пробел:
- После инициалов: `А.` + пробел → `А.` + nbsp
- Между числом и единицей: `5` + пробел + `м` → `5` + nbsp + `м`
- После №: `№` + пробел → `№` + nbsp

### Реализация
Добавить input rules в файл `src/main.js` (функция `buildInputRules`).
Или создать новый файл `src/typography-rules.js` и импортировать.
